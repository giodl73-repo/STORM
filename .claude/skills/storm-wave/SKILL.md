# STORM Wave

Use this skill to create or update STORM waves.

## Wave requirements

- Define a product-neutral weather, hazard, preparedness, or recovery goal.
- Identify affected crates and consumer repos.
- Name downstream handoff boundaries for PORTO, CERES, FAUNA, FLORA, RITE,
  BANISH, CANON, LUCIA, and TRACKER.
- Do not add real-world emergency advice or forecasting claims without a research
  pass and source custody.
- Include validation expectations for every pulse.
